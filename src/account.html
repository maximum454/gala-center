Уважаемые клиенты, с помощью пластиковой карты Вы можете оплатить:

  • Заказ товара к отгрузке, согласованный и подтвержденный с менеджером по работе с клиентам

Для оплаты Вы используете только именную пластиковую карту для ИП, либо корпаративную карту организации для ООО, ОАО
